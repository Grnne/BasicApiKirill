# BasicApi — Feature Workflow (TDD)

Minimal TDD workflow for adding features to this chat API project.

**Architecture:** `Controller → Handler → Service (optional) → Repository (in Storage)`

---

## Quick Start

| Step | File |
|------|------|
| Plan your feature | `01-feature-checklist.md` |
| Write failing test (RED) | `02-tdd-red.md` |
| Write minimal code (GREEN) | `03-tdd-green.md` |
| Refactor (REFACTOR) | `04-tdd-refactor.md` |
| Run tests | `05-commands.md` |
| Test patterns | `06-test-patterns.md` |
